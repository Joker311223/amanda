# DBT技能随身练

青少年DBT技能学习与实践平台，提供辩证行为疗法核心技能的清晰讲解、实用练习、进度追踪与个性化提醒。

## 产品介绍

DBT技能随身练是一个专为青少年设计的辩证行为疗法（DBT）学习平台，通过微信小程序的形式，为用户提供便捷的情绪管理技能学习体验。

### 核心功能

- **引导学习流程**：首次使用时提供DBT介绍和用户信息收集
- **视频课程学习**：14门核心DBT技能课程，支持进度追踪
- **作业练习系统**：18项配套作业，巩固学习效果
- **技能卡片工具**：28张技能卡片，随时查阅要点
- **我的笔记功能**：记录和追踪情绪变化
- **进度管理**：学习进度可视化，经验值激励系统

### DBT四大核心技能模块

1. **正念技能**：提高注意力和正念水平
2. **痛苦耐受技能**：帮助处理危机和困难情况
3. **情绪调节技能**：学习理解和调节情绪
4. **人际效能技能**：改善人际交往和沟通能力

## 技术架构

### 前端技术
- 微信小程序原生开发
- WXML + WXSS + JavaScript
- 响应式设计，适配不同屏幕尺寸

### 数据存储
- 微信小程序本地存储（wx.setStorageSync/wx.getStorageSync）
- 用户数据、学习进度、我的笔记本地化存储

### 主要页面结构
```
pages/
├── onboarding/          # 引导页面
├── index/              # 首页
├── learning/           # 学习中心
├── assignments/        # 作业中心
├── assignment-detail/  # 作业详情
├── tools/             # 工具页面
├── emotion/           # 我的笔记
├── video/             # 视频播放
└── skill-card/        # 技能卡片详情
```

## 安装和使用

### 开发环境要求
- 微信开发者工具
- Node.js（可选，用于项目管理）

### 快速开始
1. 下载或克隆项目代码
2. 使用微信开发者工具打开项目
3. 配置小程序AppID（在project.config.json中）
4. 点击编译运行

### 项目配置
在 `project.config.json` 中配置你的小程序信息：
```json
{
  "appid": "你的小程序AppID",
  "projectname": "DBT技能随身练"
}
```
