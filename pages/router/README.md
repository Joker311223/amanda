# Router 页面 - 学习路线图

## 页面概述

这是一个清新的绿白色调学习路线图页面，用于记录和展示用户的课程学习进度。采用时间线式的路线图设计，让学习路径一目了然。

## 设计风格

- **色调**：绿色 + 白色
- **风格**：清新、简洁、现代
- **布局**：垂直时间线路线图

## 主要功能

### 1. 顶部统计卡片
显示学习进度的关键数据：
- **已完成**：已完成的课程数量
- **总课程**：课程总数
- **完成度**：学习完成百分比

### 2. 学习路线图
以时间线形式展示所有课程：
- **路径线**：绿色渐变的垂直线条，连接所有课程
- **课程节点**：圆形节点，显示课程序号或状态图标
- **课程卡片**：展示课程详细信息

### 3. 课程状态

#### ✅ 已完成 (Completed)
- 绿色圆形节点，显示 ✓ 图标
- 卡片左侧绿色边框
- 显示完成日期

#### 🟡 进行中 (Current)
- 黄色圆形节点，带脉冲动画
- 卡片左侧黄色边框，带阴影
- 显示学习进度条

#### ⚪ 未开始 (Locked)
- 灰色圆形节点，显示课程序号
- 卡片半透明，灰色边框
- 点击提示"请先完成前面的课程"

## 交互功能

### 点击课程卡片
- **已完成/进行中**：跳转到课程视频页面 `/pages/video/video`
- **未开始**：提示"请先完成前面的课程"

### 视觉反馈
- 点击卡片时有缩放动画
- 进行中的课程节点有脉冲动画
- 结束标记有弹跳动画

## 数据来源

- `app.globalData.learningProgress` - 学习进度数据
  - `completedCourses` - 已完成课程ID列表
- `app.globalData.courses` - 课程列表
  - `id` - 课程ID
  - `title` - 课程标题
  - `category` - 课程分类
  - `duration` - 课程时长
- `coursesCompletedDates` - 本地存储的完成日期

## 样式特点

### 颜色方案
- **主绿色**：`#6BBF59` - 已完成状态
- **浅绿色**：`#A8E063` - 渐变色
- **黄色**：`#FFC107` - 进行中状态
- **灰色**：`#F5F5F5` - 未开始状态
- **背景**：`#F0F9F0` 到 `#FFFFFF` 的渐变

### 动画效果
1. **脉冲动画** - 进行中的课程节点
2. **弹跳动画** - 结束标记图标
3. **缩放动画** - 点击卡片时
4. **渐变动画** - 进度条填充

## 使用场景

1. **查看学习进度**：一眼看到已完成多少课程
2. **了解学习路径**：清晰的时间线展示学习顺序
3. **继续学习**：点击进行中的课程继续学习
4. **回顾课程**：点击已完成的课程进行复习

## 技术实现

- 使用 `scroll-view` 实现垂直滚动
- CSS 渐变实现路径线效果
- Flexbox 布局实现卡片排列
- CSS 动画实现交互效果
- 条件渲染实现不同状态展示
